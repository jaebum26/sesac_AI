걸어서 세계속으로 설명서

git에 있는 파일 zip파일로 다운로드하고 압축 풀기

미니콘다 프롬프트에서 파이썬 3.8 환경을 만든 후 프로그램 폴더 내의 install_code.py를 python install_code.py 명령어로 실행

보정과정 : 사용자는 노트북 중앙에 서서,위 숫자가 300/300이 될 때까지 웹캠 응시

음성어 장소 입력 과정 : 녹음이 시작된다는 메시지가 출력되면 원하는 지역(예: 경복궁)을 말해서 위치로 이동

스트릿뷰 로딩 후, 사용자의 시선에 따라 뷰의 시점 이동

음성 명령 입력 : 추가적인 음성 명령어 입력 시 해당 프로세스 실행
명령어 목록: 보정, 그림 시작, 그림 종료, 지워 줘, 사진
(예: "그림 시작" 명령어 실행 시 스트릿 뷰 배경에 그리기 시작)

음성 명령어 "사진" 입력 또는 손으로 손흥민 세레머니 포즈를 취할 시, 스트릿 뷰 화면 캡쳐 및 병합

"지워 줘" 명령어 실행 시 그림 전체 삭제
